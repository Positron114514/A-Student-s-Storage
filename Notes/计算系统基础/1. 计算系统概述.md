## 1.2 图灵机

由机器头和纸带 (小方格) 组成

两种简单的动作:

- 在纸上写 / 擦除一个符号
- 移动一个单元格

使用内部的action table, 将一组数值变为另一组数值

**程序表**

- q -> 当前状态
- b -> 当前方格原符号
- a -> 修改后符号
- m -> 机器头移动方向
- q' -> 下一状态

**Example: 计算 $f(x) = x + 1$**
