# 4.1 RS锁存器

![[../00 Resource/res/Pasted image 20240319163635.png]]

工作原理:

R: Reset 复位

S: Set 置位

- R = S = 1: 存储当前状态
- R = 1, S = 0: a设为0, 写入0
- R = 0, S = 1: a设为1, 写入1
- R = S = 0: SB

## 门控D锁存器

![[../00 Resource/res/Pasted image 20240319170932.png]]
WE: 是否允许写入

D: 写入什么

将RS锁存器并联并用一个 D WE来控制, 组成寄存器

# 4.2 状态

指在某一特定时刻, 系统内所有相关部分的一个瞬态图

寄存器是有限的 $\to$ 状态是有限的

有限状态机由五个元素组成:

- 有限数目状态
- 有限数目外部输入
- 有限数目的输出
- 状态转换函数
- 输出函数

可以用状态图表示

## 时钟

用于触发状态机向下一个状态转换的机制

时钟发出的值在 0 1 之间交替, 时钟周期为一个完整的 0  1 所用的时间

Example: 信号灯
