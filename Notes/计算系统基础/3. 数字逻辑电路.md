# 3.1 二进制逻辑运算

位组合编号: 最右侧为 [0] 

AND OR NOT XOR

in C:

- `&` AND
- `|` OR
- `~` NOT
- `^` XOR
- `<<` left 右侧用 0 填充
- `>>` right 算数右移

```c
int a = 3; // 0011
int b = 4; // 0100

int c = a & 1; // 0001, 一种检测奇偶的方式
c = ~a; // 1100
c = a ^ b; // 0111
c = a << 2; // 1100 = a * 2^2
c = a >> 1; // 0001
```

掩码: 做按位与运算

example: 取后两位: & 3 (11)
