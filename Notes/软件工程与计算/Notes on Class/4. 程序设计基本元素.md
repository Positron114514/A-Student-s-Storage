# print 和 None

`None` 在python中代表 Nothing

一个函数如果没有返回值, 则其返回值默认为 `None`

`None` 不会作为一个表达式的值被解释器解释

**纯函数和非纯函数**

- 纯函数: 只返回值
- 非纯函数: 有一些额外的作用

`print` 的返回值为 `None`

```python
>>> print(print(1), print(2))
1
2
None None
```

# 4.3 函数设计

**规范命名:** 驼峰法 or 下划线

函数名称要额能体现函数的作用, 少用 `1` `l` `I`

好的函数设计: 

- 每个函数只做一个任务
- 少重复
- 更具一般性
